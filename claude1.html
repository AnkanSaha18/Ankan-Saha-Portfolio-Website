Optimizing Hyperledger Framework for High Transactions Throughput in Private Blockchain

Technologies: Hyperledger Fabric, Node.js

Developed an innovative approach to optimize transaction throughput in Hyperledger private blockchain networks, achieving significant improvements in speed and efficiency through enhanced network architecture, advanced chaincode functionality, and seamless Node.js integration.

My thesis focused on enhancing the performance of a private blockchain network through meticulous optimization of several key network parameters. This included adjustments to block size, the number of channels, transaction volumes, and the endorsing policy. The goal was to increase throughput while simultaneously reducing network latency significantly.
